# IoT Anomaly Detection

## Description

This project detects anomalies in IoT systems using a hybrid deep learning approach. It analyzes data generated from IoT environments to identify unusual patterns and potential system-level issues.

## Tool Used

* Google Colab
* Python

## How to Run

Open the notebook in Google Colab and run all cells.

## Note

All preprocessing, model building, and results are included in the notebook.
